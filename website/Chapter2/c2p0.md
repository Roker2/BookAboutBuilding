# Введение

Если Вы имеете аккаунт на GitHub или на чем-то подобном и знаете git, то это глава Вам не нужна.

Начнем с определения "git".

::: info
Git - распределенная система управления версиями.

© Wikipedia
:::

git позволяет создавать историю изменений и манипулировать ею, тем самым достигая возможности более быстро разбираться в проблеме, анализировать код, более эффективно вести совместную разработку и т. д. Без знаний git'а будет очень сложно работать с исходниками Android и в принципе в IT-сфере, так как инструмент ходовой, бесплатный и очень полезный. Пренебрегать им нельзя ни в коем случае! Если был опыт работы с Mercurial'ом, то освоить git будет проще.

Обычно названия коммитов в дереве устройства и в вендоре имеют такие название:

1. `codename: part_of_tree: Name` *(пример: "mido: mixer_paths: Remove duplicate entries")*
2. `codename: Name` *(пример: "mido: Kang tissot s5k5e8_ofilm")*
3. `part_of_tree: Name` *(пример: "power: Handle Mode::LAUNCH power hints")*
